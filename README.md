# EIM Widget Library

An interactive collection of educational electronics and engineering widgets developed by EIM Technology. This library is designed for STEM education, embedded systems learning, circuit analysis, and interactive demonstrations for students, educators, and makers.


## Live Demo

https://eim-widget-library.vercel.app/

---

# Features

- Interactive engineering and electronics widgets
- Modern responsive web interface
- Educational focused UI/UX
- Lightweight and fast loading
- Designed for classroom and self learning environments
- Compatible with desktop and mobile browsers
- Easy to expand with additional widgets

---

# Current Widgets

Some examples of widgets included in the library:

- Ohm's Law Calculator
- Resistor Color Code Calculator
- RC Circuit Simulation
- Electronics Formula Helpers
- Educational Visualization Tools
- Interactive STEM Learning Components

More widgets are continuously being added.

# Local Development

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPOSITORY.git
```

Open the project folder:

```bash
cd YOUR_REPOSITORY
```

Run locally using VS Code Live Server or any local web server.

Example using VS Code:

1. Install the "Live Server" extension
2. Right click `index.html`
3. Click "Open with Live Server"

---

# Deployment

This project is deployed using Vercel.

To deploy your own version:

1. Fork or clone this repository
2. Push to your GitHub account
3. Import the repository into Vercel
4. Deploy instantly
